## Your First Picker

Create a color picker in three simple steps:

### 1. Import the Module

```js
import 'color-input'
```

### 2. Add the Element

```html
<color-input value="oklch(70% 0.2 240)" colorspace="oklch"></color-input>
```

### 3. Try It

<ColorInputDemo value="oklch(70% 0.2 240)" colorspace="oklch" />

That's it! The picker is fully functional with keyboard support, touch gestures, and automatic gamut detection.

## Understanding the Output

The `value` attribute accepts standard CSS color syntax. When you change the color, the component emits a `change` event with the updated value:

```js
const picker = document.querySelector('color-input');

picker.addEventListener('change', (event) => {
  console.log(event.detail.value);      // e.g., "oklch(65% 0.25 240)"
  console.log(event.detail.colorspace); // e.g., "oklch"
  console.log(event.detail.gamut);      // e.g., "srgb" or "p3"
});
```

## Basic Interactions

- **Click/Tap** the trigger button to open the picker
- **Drag** sliders to adjust channels
- **Arrow keys** ↑↓ increment/decrement by 1, Shift+↑↓ by 10
- **Enter** or **Space** on the trigger opens the popover
- **Escape** closes the popover
- **Tab** navigates between controls

## Opening the Picker Programmatically

You can control the popover via the Popover API button command:

```html
<color-input id="picker"></color-input>
<button commandfor="picker" command="toggle-popover">Toggle Picker</button>
```

<div class="demo">
  <div class="demo-row">
    <color-input id="demo-picker" value="oklch(60% 0.15 320)" colorspace="oklch"></color-input>
    <button commandfor="demo-picker" command="toggle-popover">Toggle Picker</button>
  </div>
</div>

Or use the `show()` and `close()` methods:

```js
const picker = document.querySelector('color-input');

// Open
picker.show();

// Close
picker.close();
```
