## Features

<CardGrid>
  <Card title="Wide-Gamut Colors" icon="star">
    sRGB, Display P3, Rec2020, and beyond. Automatic gamut detection shows which space contains your color.
  </Card>
  <Card title="Multiple Color Spaces" icon="palette">
    HSL, HWB, LAB, LCH, OKLCH, OKLAB, and wide-gamut RGB spaces with seamless conversion.
  </Card>
  <Card title="Area Picker" icon="open-book">
    2D color area that adapts its axes to the active color space. Pointer drag and keyboard arrow-key support.
  </Card>
  <Card title="CSS Parts" icon="puzzle">
    Style every part of the picker with `::part()` selectors — trigger, panel, sliders, and more.
  </Card>
</CardGrid>

## Quick Start

<Tabs>
  <TabItem label="Usage">
    ```html
    <script type="module">
      import 'https://cdn.jsdelivr.net/gh/argyleink/css-color-component/dist/index.js'
    </script>

    <color-input value="oklch(75% 30% 180)"></color-input>
    ```
  </TabItem>
  <TabItem label="Result">
    <ColorInputDemo
      value="oklch(75% 30% 180)"
      showCode={false}
    />
  </TabItem>
</Tabs>

:::note
The custom element auto-registers on import. Use it in any HTML document or framework. The [Popover API](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API) is required (Chrome 114+, Safari 17+, Firefox 125+).
:::
